Archivos Aceptados por WebMinerX

ARFF

La información se organiza en dos secciones:
En la Sección de Encabezado (Header) se proporciona información sobre los atributos y la relación entre ellos. En la Sección de Datos (Data) se proporcionan los datos en sí separados por comas.

CSV

Es un formato de archivo que se utiliza para almacenar datos tabulares, como una hoja de cálculo o una base de datos, donde cada línea del archivo representa una fila de datos y los valores de cada columna están separados por comas.

Ejemplo de archivo ARFF

    @relation Ejemplo
    @attribute Dato1 real
    @attribute Dato2 real
    @attribute Dato3 real
    @data
    1,1,0
    2,1,1
    1,2,0
    2,2,1
    4,1,3
    4,2,1
    4,3,3
    4,4,5
    2,4,1
    3,4,4
    2,5,2
