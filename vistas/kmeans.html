Algoritmo K-Means

El algoritmo K-Means es un algoritmo de aprendizaje no supervisado, que se encarga de agrupar datos en K grupos, donde K es un número entero positivo. El algoritmo K-Means es un algoritmo iterativo, que se encarga de minimizar la suma de las distancias de cada objeto al centroide de su grupo o cluster.
